## Color Coding Reference

- **[GREEN]** - Safe paper trading mode, informational messages
- **[INFO]** - Informational messages, status indicators  
- **[WARNING]** - Live trading warnings, caution messages
- **[ERROR]** - Critical warnings, requires attention

## User Flow

1. **Start in Paper Mode** (default)
   - Green banner
   - Paper radio selected
   - Safe to connect

2. **Select Live Mode**
   - Shows warning panel
   - Requires two confirmations
   - Button disabled until confirmed

3. **Confirm Switch**
   - Click both checkboxes
   - Click "Switch to LIVE Trading"
   - Mode changes to live
   - Banner turns red

4. **Switch Back to Paper**
   - Select paper radio
   - Simple confirmation
   - Click "Switch to PAPER Trading"
   - Banner turns green
