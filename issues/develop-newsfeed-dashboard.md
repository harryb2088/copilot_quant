## Issue: Develop a Newsfeed/Dashboard Page

### Overview
Create a newsfeed/dashboard page that provides an overview of global market events and key news updates for the day. This page will include:

1. **Overnight Market Performance**  
   - Major indices (e.g., S&P500, NASDAQ, DAX, Nikkei, etc.)
2. **Major Financial or Economic News**  
   - Central bank announcements, geopolitical events, etc.
3. **Commodities Prices**  
   - Prices for crude oil, gold, etc.
4. **Currency Pair Movements**  
   - Movements in pairs such as USD/EUR, USD/JPY, etc.
5. **Upcoming Key Events**  
   - Events like earnings, economic data releases (CPI, non-farm payrolls, etc.)

### Features Required
- A simple and clear layout for quick reading.
- Interactive and live updates for prices (optional but desirable).

### Integrations Needed
- Add necessary integrations for market data and news APIs (e.g., Alpha Vantage, Bloomberg, etc.).

### Deliverable
- Provide instructions for developers to integrate and use the dashboard, including any needed API keys or configuration.
